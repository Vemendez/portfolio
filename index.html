World Libraries (Tableau)

This project showcases a comprehensive Tableau dashboard that visualizes data on libraries around the world. The dashboard includes three key views:

  • A bar chart illustrating the total number of libraries by region.
  • A crosstab detailing expenditures, total libraries, and total users by country.
  • A global map view highlighting the distribution of libraries by country, with an interactive filter by region.
The data visualization provides valuable insights into the global distribution and expenditure on libraries, helping stakeholders understand regional differences and resource allocation.
